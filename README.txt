Synchronization Program

External libraries used:
sqlalchemy
tkcalendar
APScheduler

Command Prompt Options:
      1 - Synchronization Folders Options -> Register folder pairs and view existing pairs
      2 - Synchronization Logs Options    -> Display and export synchronization logs
      3 - Schedules Options               -> Display and set schedules 
      4 - Synchronize Folders             -> Manually synchronize folders
      5 - Launch GUI                      -> Launch GUI interface
      C - Close                           -> Close the program

GUI Options:
      Set Synchronization Schedule    -> Set synchronization schedules
      View Schedule Synchronization   -> Display existing schedules
      View Synchronization Logs       -> Display and export synchronization logs
      Set Synchronization Folders     -> Register folder pairs and view existing pairs
      Synchronize Folders             -> Manually synchronize folders selected in the combobox
      Close                           -> Close GUI interface
